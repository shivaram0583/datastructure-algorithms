# Behavioural Round — JP Morgan Round 3

## Overview
Round 3 at JP Morgan is a 45-60 minute conversation with a senior engineer or manager. It evaluates leadership, ownership, collaboration, and culture fit.

## JP Morgan Core Values
- **Integrity** — Do the right thing
- **Client Focus** — Long-term client interests
- **Accountability** — Own outcomes
- **Collaboration** — Cross-functional teamwork
- **Excellence** — High standards always

## Quick Tips
- Use "I" not "we" — interviewers want to know YOUR contribution
- Quantify every result (%, time saved, latency, error rate)
- Prepare 5-6 stories that cover multiple question types
- End every answer with what you LEARNED or would do DIFFERENTLY
- Relate answers to JP Morgan's scale and financial domain when possible
